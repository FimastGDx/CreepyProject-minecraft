# CreepyProject-minecraft
Minecraft new creepy versions launcher, CreepyLauncher new fork

# Build
Required NodeJS 23.
```cmd
npm install
```
```cmd
npm run make
```
```cmd
:: DEV mode
npm run start
```
# How to install
Download latest versions at Releases, run .exe and wait for install
